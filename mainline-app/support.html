What GitHub token permissions do I need?

A classic Personal Access Token with repo scope, or a fine-grained token with Issues: Read and write on the target repository.

The app won't connect to my repo

Check that the token is valid and has not expired. In GitHub → Settings → Developer settings → Personal access tokens, verify the token and its scopes. Then open Mainline → Settings → Disconnect and reconnect with the updated token.

Tasks aren't syncing

Pull to refresh in the app, or wait for the automatic sync interval. Make sure your Mac has internet access and the token hasn't been revoked.

How do I use the AI quick-add?

In Settings, enter a DeepSeek API key. Then type a natural-language task in the quick-add bar (e.g. "finalize pitch deck p0 this week") and press Enter. The AI parses it into a structured task before creating the GitHub Issue.

How do I disconnect and remove my data?

Open Settings → Disconnect. This removes your tokens from Keychain and clears the repo configuration. You can also revoke the token directly from your GitHub account settings.
