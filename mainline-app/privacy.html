Mainline — Privacy Policy

Effective date: 2026-04-21

Mainline ("the app") is a macOS task management app that connects to your own GitHub repository. Your tasks live in your GitHub repo; Mainline never touches a server it controls.

1. Data we do not collect

We do not collect any personal data. The app does not create an account on our side, ask for your email, send data to our servers, or include any analytics, crash reporting, advertising, or attribution SDKs.

2. What the app connects to

  • GitHub API (api.github.com) — to read, create, update, and close issues in the repository you specify. All requests use the Personal Access Token you provide, stored in your Mac's Keychain.
  • DeepSeek API (api.deepseek.com) — optionally, if you provide a DeepSeek API key, the app sends your task input text to DeepSeek for AI-powered parsing. This key is also stored in Keychain. This feature is opt-in.

No other network connections are made.

3. What is stored locally on your Mac

  • Your GitHub Personal Access Token (Keychain, encrypted by macOS).
  • Your DeepSeek API key, if provided (Keychain, encrypted by macOS).
  • Your repo owner and repo name (UserDefaults).
  • App preferences — sync interval, onboarding-completed flag.

None of this is transmitted to us. Uninstalling the app removes its sandboxed storage; API keys can also be removed via Settings → Disconnect inside the app.

4. Third parties

The only third-party services the app contacts are GitHub and DeepSeek, using credentials you explicitly provide.

  • GitHub Privacy Statement
  • DeepSeek Privacy Policy

5. Children

The app is not directed at children under 13 and does not collect any personal data.

6. Your rights

You can disconnect and remove all local data from Settings → Disconnect, or revoke the GitHub token from your GitHub account settings at any time. Because we collect no data on our side, there is nothing for us to export or delete.

7. Changes

If this policy changes materially, we will update the effective date and post the new version at this URL.
